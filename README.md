Step 1: Clone the repo

```bash
git clone https://github.com/sahil-009/Chat-Application.git
```

Step 2: cd into the cloned repo and run:

```bash
npm install
```

Step 3: Start the chat app (development mode)

```bash
npm run dev
```

Step 4: Start the chat app

```bash
npm start
```
